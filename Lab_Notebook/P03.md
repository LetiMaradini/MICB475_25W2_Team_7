# P03 - Phyloseq and Rarecurve

Feb 27th, 2026

## Purpose:
To transfer the files from the servet to individual computer to run on Rstudio. To generate the phyloseq object, filter to remove ASV's with counts fewer than 5 and samples with fewer than 100 reads, and rarify phyloseq object to a chosen sampling depth.

## Material: 
1. QIIME2
2. R & Rstudio (packages: phyloseq, ape, tidyverse, vegan)
3. microgravity_metadata.tsv
4. feature-table.txt (from table-no-mitochondria-no-chloroplast)
5. taxonomy.tsv
6. tree.nwk

## Method:
1. Connect to Cisco Anyconnect Secure VPN. Open the terminal and login to MICB 475 class server using the provided login credentials.
2. Move all files to be exported to a folder, convert .biom file into .txt (from table-no-mitochondria-no-chloroplast) and export
* In RStudio
4. Format OTU table, sample metadata, and taxonomy
5. Create phyloseq object by merging formatted OTU table, sample metadata, taxonomy, and phylogeny tree
6. Filter to remove ASVs that have less than 5 counts total and samples with less than 100 reads
7. Rarify to a sampling depth of 64,411.
